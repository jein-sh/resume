# Resume — Vue 3 + Vite

Это одностраничное приложение-резюме, созданное с использованием **Vue 3** и **Vite**.

## Технологии

- [Vue 3](https://vuejs.org/)
- [Vite](https://vitejs.dev/)

## Установка

npm install

## Запуск проекта для разработки

npm run dev

## Сборка для продакшена

npm run build
